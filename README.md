# NUShare
A common web portal for students to share resources (Computer-Human Interaction project)

## Installation
```bash
git clone https://github.com/nakulvr/HCI_PROJECT
npm install
node app.js
```

## Technology Used
![Back-end:]
1. Node.js
2. MongoDB
3. Express.js
4. Bootstrap
5. HTML
6. CSS
