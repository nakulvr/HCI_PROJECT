# NUShare
A common web portal for students to share resources (Computer-Human Interaction project)

## Installation
Pre-requisite is to have the “mongod” service running
```bash
git clone https://github.com/nakulvr/NUShare
npm install
node app.js
```

## Technology Used

### Back-end:
1. Node.js
2. MongoDB
3. Express.js

### Front-end:
1. Bootstrap
2. HTML
3. CSS
